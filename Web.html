Web (HTML & CSS)

Pour créer un site web, on utilise deux langages complémentaires :

- HTML (HyperText Markup Language) permet de structurer le contenu d'une page : texte, titres, images, liens, listes, tableaux, etc.
- CSS (Cascading Style Sheets) permet de mettre en forme ce contenu : couleurs, polices, mise en page, alignement…

Une page web commence toujours par une structure de base : 
<!DOCTYPE html>
<html>
  <head> ... </head>
  <body> ... </body>
</html>

Le contenu visible de la page se place dans <body>.
On utilise des balises pour organiser les éléments. Certaines balises contiennent du contenu (<p>, <h1>, <a>, etc.), d'autres sont orphelines (<br />, <img />).

Les liens se créent avec <a href="...">, les images avec <img src="..." alt="..." />.
Les listes peuvent être à puces (<ul>) ou numérotées (<ol>), et les tableaux sont construits avec <table>, <tr>, <td>, et <th>.

HTML5 a introduit des balises sémantiques pour mieux organiser les pages :
<header> (en-tête), <footer> (pied de page), <nav> (menu), <section>, <article>, <aside>.
