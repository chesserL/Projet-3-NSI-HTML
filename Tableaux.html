Représentation des données : les tableaux

I. Introduction

Jusque-là, nous avons appris à travailler avec un nombre limité de variables. Cependant, il peut arriver dans un programme de devoir lire, stocker, traiter et imprimer des dizaines voire des centaines ou encore des milliers de valeurs.

II. Comment stocker ces données ?

Une première option est l’utilisation de tableau (ou bien le type list en Python) :
En informatique, un tableau est une structure de données représentant une séquence finie d’éléments auxquels on peut accéder efficacement par leur indice, dans la séquence. En Python, les tableaux sont représentés par le type List. Pour définir un tableau, on utilise les crochets :
Pour créer un tableau vide, on écrit :
     >>> tab = [ ]
     On peut créer par extension un tableau avec des valeurs :
>>> tab = [1, 2, 3, 4]

III. Comment accéder à ces données ?

Les éléments d'un tableau peuvent être comparés à des éléments d'une séquence : chaque élément a une position précise dans le conteneur, comme nous l'avons vu pour les chaînes de caractères (String). On appelle cette position indice, c'est cet indice qui nous permet d'accéder à un élément en particulier. Le premier élément d'un tableau a pour indice 0.
>>> tab = [1, 5, 7, 3, 2, 9]
>>> tab[0]
1
>>> tab[4]
2
Pour connaître le nombre total d’éléments dans un tableau, on utilise la fonction len :
    >>> tab = [‘a’, ‘e’, ‘b’, ‘t’]
    >>> len(tab)
    4
On peut désigner une tranche en indiquant l'indice du premier élément, ainsi que celui du dernier non inclus :
    >>> tab = [1, 5, 7, 3, 2, 9]
    >>> tab[1:4]
    [5, 7, 3]

V. Quelques propriétés

Une structure comme un tableau peut être modifiable : on parle d'un objet mutable.
• Il est possible d’ajouter un élément en fin de séquence avec l’utilisation de la fonction append.
>>> t = [ 1, 2 ]
>>> t.append( 3 )
>>> print( t )
[1, 2, 3]
• Il est possible de modifier un élément en particulier, on lui affecte une nouvelle valeur.
>>> t = [ 1, 12, 3]
>>> t[ 1 ] = 2
>>> print( t )
[1, 2, 3]
• Il est préférable de ne stocker que des données de même type dans un tableau.
• Il peut-être passé comme paramètre d'une fonction.

A retenir:

Opération Résultat
x in t Renvoie True si un élément de t est égal à x
x not in t Renvoie True si aucun élément de t n'est égal à x, False sinon
len(t) Renvoie le nombre d'éléments de t
t == t1 Renvoie True si t et t1 sont de même type, ont la même longueur, et ont des éléments égaux 2 à 2
t[i] Renvoie l'élément d'indice i de t
t[i:j] Renvoie une partie de t de l'indice i à j non inclus
t.index(x) Renvoie l'indice de la première apparition de x dans t
t.count(x) Renvoie le nombre d'apparitions de x dans t
s + t Renvoie une nouvelle séquence qui correspond à la concaténation de s et de t (seulement si s et t ont le même type)
t.append(x) Ajoute l'élément x à la fin du tableau t
t[i] = x Modifie le tableau et affecte la valeur x à la case d'indice i
t.insert(i, x) Insère l'élément x à la position de l'indice i
t.remove(x) Supprime le premier élément dont la valeur est égale à x
t.pop() Enlève du tableau l'élément qui se situe à la fin et renvoie sa valeur
