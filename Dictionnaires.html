Dictionnaires

I. Introduction

Afin de stocker des informations dans une seule et même variable, nous utilisons communément le n-upplet ou le tableau.

Par exemple :
    t = (‘Dupond’, ‘Alice’, 171, 16)

Ce n-uplet semble stocker des informations sur une personne. Le problème vient du type d’information contenue dans cette structure qui n’est pas indiqué. On peut facilement deviner qu’il s’agit d’un nom, d’un prénom, d’une taille et d’un âge. Mais sans la confirmation du développeur du code cela n’est que supposition. De plus, l’accès aux éléments se fait par numéro de colonne et non par type d’information.

II. Définition

A la différence d’un n-uplet, le dictionnaire permet d’associer à chaque valeur une clé (unique). Cette clé fonctionne comme un indice, en effet lorsque l’on voudra accéder à une valeur du dictionnaire il faudra l’appeler par sa clé (Et non par sa position).
De plus la clé peut être une chaîne de caractère (str) ou un entier (int)

Voici ce que cela peut donner lorsque l’on reprend l’exemple du dessus avec un dictionnaire.

 d = {'nom': 'Dupond', 'prenom': 'Alice', 'age': 16, 'taille': 171}

Ici chaque valeur correspond à une clé. Et donc, la structure correspond mieux au stockage de ce type de donnée.

III. Découverte du type dict

La création d’un dictionnaire peut se faire de 3 manières différentes.

# Création d'un dictionnaire vide :
d = dict()
d2 = {}
# Création d'un dictionnaire avec données :
d3 = {"nom": "Dupont", "prenom": "Alice", "age": 16, "taille": 171}

 L’accès aux données se fait par la clé qui lui est associée. La modification quant à elle se fait par l’accès de la donnée suivie de la nouvelle valeur.

# Accès à la donnée :
>>> d3 = ["nom"]
"Dupont"
# Modification :
>>> d3["nom"] = "Timoleon"

Pour ajouter un élément il suffit d’appeler le dictionnaire suivit de la clé associée (nouvelle), puis d’y affecter la nouvelle valeur.

>>> d3 = ["classe"] = "première"
>>> d3
{"nom": "Timoleon", "prenom": "Alice", "age": "16", "taille": 171, "classe": "première"}

 Le parcours d’un dictionnaire est possible à l’aide d’une boucle for (le for permet de parcourir les clés du dictionnaire).

d3 = **{"nom": "Timoleon", "prenom": "Alice", "age": "16", "taille": 171, "classe": "première"}**
for c in d3:
    print("La clé", c, "est associée à la valeur", d3[c])

 Il existe des méthodes associées aux dictionnaires :

# Récupération des clés :
d3.keys()
# Récupération des valeurs :
d3.values()
# Récupération des items :
d3.items()

Ces fonctions renvoient un élément de type 'class dict_keys', 'class dict_values', 'class dict_items' difficilement manipulable. Il est donc conseillé de changer ce type par le type list (tableau python).
