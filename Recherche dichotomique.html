Recherche dichotomique

Explication

C'est une méthode de recherche d'un nombre basé sur le principe de " la moitié de la moitié ". Par exemple, si on cherche un nombre entre 1 et 100, on va dire 50. Si c'est plus, on va dire 25, si c'est moins 75, et ainsi de suite jusqu'à trouver le bon nombre.

Code python

def recherche_dichotomique(tab,el):
     deb=0
     fin=len(tab)-1
     while deb<=fin:
         milieu=(fin+deb)//2
         if tab[milieu]==el:
             return True
         elif tab[milieu]>el:
             fin=milieu-1
         else:
             deb=milieu+1
     return False
