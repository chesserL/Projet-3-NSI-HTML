Boucles

I. La boucle bornée : for

Une boucle 'for' permet de répéter un bloc d'instructions, un nombre fini de fois. Une boucle 'for' définit une variable qui prend successivement toutes les valeurs de la séquence parcourue.
Voici la structure d'une instruction for :

for var in sequence:
    bloc d'instructions
La fonction en Python range permet de définir une intervalle. Cette dernière est utile lorsqu’une boucle for repose sur l’utilisation d’une séquence de nombre d’entiers.
La fonction range peut recevoir 3 paramètres : range(deb, fin, pas)

    • deb → début de l’intervalle (inclus)
    • fin → fin de l’intervalle (exclus)
    • pas → l’incrémentation entre 2 éléments de l’intervalle

II. La boucle non bornée : while

Le principe de la boucle while est d’exécuter un bloc d’instructions tant qu’une condition donnée est vrai.
Voici la structure d’une instruction while :

while condition :
    bloc d'instructions

Exemple de boucle while :

tmp = 0
while tmp <= 10:
    print(tmp)
    tmp += 1

La boucle permet d’afficher les nombres entre 0 et 10 inclus
Attention ! La deuxième ligne dans le bloc d’instructions est primordiale car elle assure la terminaison de la boucle. Sans cette incrémentation, on obtient une boucle infinie étant donné que la condition sera toujours à vrai.
Lorsque l’on souhaite utiliser une boucle while, on définit une variable, au préalable, qui va servir de compteur.

Pour finir, on utilisera une boucle bornée lorsque l’on sait combien de fois on veut réaliser une action. On utilisera donc une boucle non bornée pour réaliser des comportements particuliers.
