Boucles

I. La boucle bornée : for

Une boucle 'for' permet de répéter un bloc d'instructions, un nombre fini de fois. Une boucle 'for' définit une variable qui prend successivement toutes les valeurs de la séquence parcourue.
Voici la structure d'une instruction for :

for var in sequence:
    bloc d'instructions
La fonction en Python range permet de définir une intervalle. Cette dernière est utile lorsqu’une boucle for repose sur l’utilisation d’une séquence de nombre d’entiers.
La fonction range peut recevoir 3 paramètres : range(deb, fin, pas)
    • deb → début de l’intervalle (inclus)
    • fin → fin de l’intervalle (exclus)
    • pas → l’incrémentation entre 2 éléments de l’intervalle
