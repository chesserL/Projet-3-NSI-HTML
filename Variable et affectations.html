I.Variable et affectation

Une variable est une boite contenant un objet. L'affectation donne une valeur à une variable. Exemple d'affectation : a = 5 Ici, la variable se nomme a et contient la valeur 5

II.Types de bases sous Python

II – Types de bases sous Python Il existe plusieurs types de données en informatique. Le type définit la nature des valeurs que peut prendre une donnée. Pour commencer, nous rencontrerons principalement les types suivants : • le type int (integer / entier) • le type float (les nombres décimaux) • le type str (string / chaîne de caractère) " " / Je suis un texte 'Je suis un autre texte' Une chaîne de caractère est caractérisé par l’utilisation de guillemet. Il est possible que Python nous donne le type d’une variable grâce à une commande nommé type()

III – Opérations élémentaires

Le = permet de faire une affectation Le + permet d’additionner Le – permet de soustraire Le * permet de multiplier Le / permet de faire une division simple Le // permet d’avoir le quotient d’une division euclidienne Le % permet d’avoir le reste d’une division euclidienne Le ** permet de faire des puissances
