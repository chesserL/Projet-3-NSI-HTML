Tri de nombres dans l'ordre croissant

I. Méthode 1:Tri par sélection

Cette méthode de tri consiste à intervertir le nombre le plus petit avec celui en première place, puis de le faire pour la deuxième place, et ainsi de suite. Voici un algorithme python le faisant :

II. Méthode 2:Tri par insertion

Cette méthode de tri consiste à décaler successivement le 2ème élément avant le premier s'il est plus petit, puis à placer le 3ème élément vis-à-vis des 2 premiers éléments, etc... Voici un algorithme python le faisant :

def decalage_pour_tri_insertion(tab,i): v=tab[i] while i > 0 and tab[i-1] > v: tab[i]=tab[i-1] i-=1 tab[i]=v def tri_par_insertion(tab): for i in range(1,len(tab)): decalage_pour_tri_insertion(tab,i) return tab
